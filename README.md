# 🛒 Quantium-Chips-Category-Analysis

A comprehensive retail analytics project for Quantium, analyzing customer purchasing behavior in the chips category and conducting controlled store trials to validate strategic recommendations. This project demonstrates end-to-end data analysis capabilities from data cleaning to experimental design and business recommendations.

📊 **[View Strategic Presentation →](https://github.com/Henryolabode/Quantium-Chips-Category-Analysis/blob/main/QVI_Strategic_Report.pptx)** *(Download to view)*

---

## 📋 Project Overview

**Client:** Quantium Retail Analytics  
**Stakeholder:** Julia - Category Manager, Chips Division  
**Dataset:** 264,836 transactions | 71,287 customers | 12-month period  

This project delivers actionable insights for chips category optimization through:
- Customer segmentation and behavior analysis
- Store trial evaluation with statistical testing
- Data-driven recommendations for 5-8% category growth
